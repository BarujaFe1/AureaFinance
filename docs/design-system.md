# Design System

## Direção visual

- premium
- minimalista
- calmo
- muito espaçado
- forte legibilidade numérica
- claro/escuro igualmente bons

## Tokens

- **fonte**: Geist Sans
- **base**: neutros próximos de zinc/slate
- **accent**: bege suave e discreto
- **radius**: 1rem a 1.25rem
- **sombra**: difusa, baixa agressividade
- **números financeiros**: tabular nums

## Regras de interface

- cards sempre com bastante respiro
- tabelas com poucas linhas visuais e foco no conteúdo
- cor forte só para alertas e CTA principal
- empty states claros e sem ruído
- dark mode sem contraste lavado
