# Roadmap sugerido

## O que está na V1

- contas
- transações
- cartões
- faturas
- parcelamento
- recorrências
- calendário financeiro
- fechamento mensal
- patrimônio resumido
- importação auditável

## Próximos passos sugeridos

1. filtros avançados em tabelas
2. edição inline de registros
3. subcategorias completas no CRUD
4. exportação CSV/JSON
5. backup agendado no próprio app
6. lembretes com notificação local
7. PWA real com cache offline
8. sincronização opcional futura

## O que deixar para V2

- integração bancária
- autenticação cloud
- multiusuário
- investimentos avançados com ativos por ticker
- OCR de comprovantes
- IA local opcional de categorização
