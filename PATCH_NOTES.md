# Aurea Finance fixpack

Este pacote foi gerado sobre a base reconstruída a partir do `project-scan-lite.md`, porque o `.rar` do repositório real não pôde ser extraído neste ambiente.

Arquivos aqui espelham caminhos reais do projeto e estão prontos para sobrescrever a raiz do repo.

Inclui:
- hotfixes de importação/workbook Money;
- saneamento de texto/UTF-8 em telas principais;
- deduplicação lógica de cartões na leitura;
- projeção por conta mais conservadora e coerente;
- materialização de recorrências até o horizonte configurado;
- fluxo mínimo de fechamento mensal;
- script doctor;
- script idempotente de reparo de mojibake legado;
- queries SQL de auditoria.
